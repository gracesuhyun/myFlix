How to Use This API


This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.
Description URL HTTP Method Query Parameter Request Response
Return a list of all movies to the user /movies GET None None A JSON object holding data about all movies
Return data about a single movie by title to the user /movies/[Title] GET :Title None A JSON object holding data about a single movie, structured like: 
						
{
	Title: '',
    Description: '',
    Genre: {
        name: '',
        description: ''
    },
    Director: {
        name: '',
        description: ''
    },
    ReleaseDate: '',
    Actors: [],
    ImagePath: '',
    Featured: ''
}
Return data about a genre by name /genre/[Genre] GET :Genre None A JSON object holding data about the selected genre, structured like: 
						
{
	name: '',
	description: ''
}
Return data about a director by name /directors/[Director] GET :Director None A JSON object holding data about the selected director, structured like: 
						
{
	name: '',
	description: ''
}
Allow new users to register /users POST None A JSON object holding data just created by user, structured like: 
						
{
	Name: '',
	Username: '',
	Password: '',
	Email: '',
	Birthday: '',
	FavoriteMovies: []
}
A JSON object holding data just created by user with newly added ID, structured like: 
						
{
	_id: '',
	Name: '',
	Username: '',
	Password: '',
	Email: '',
	Birthday: '',
	FavoriteMovies: []
}
Allow users to update their user info (such as their name) /users/[Username] PUT :Username A JSON object holding data with user info, structured like: 
						
{
	Name: '',
	Username: '',
	Password: '',
	Email: '',
	Birthday: '',
	FavoriteMovies: []
}
A JSON object holding data with updated user info, structured like: 
						
{
	_id: '',
	Name: 'updated',
	Username: 'updated',
	Password: 'updated',
	Email: 'updated',
	Birthday: 'updated',
	FavoriteMovies: []
}
Allow users to add a movie to their list of favorites /users/[Username]/movies/[Title] POST :Username, :Title None A message indicating whether movie was successfully added to user's list.
Allow users to remove a movie from their list of favorites /users/[Username]/movies/[Title] DELETE :Username, :Title None A message indicating whether movie was successfully removed from user's list.
Allow existing users to deregister /users/[Username] DELETE :Username None A message indicating whether user was successfully deleted.
