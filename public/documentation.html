How to Use This API


This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.
Description URL HTTP Method Query Parameter Request Response
Return a list of all movies to the user /movies GET None None A JSON object holding data about all movies
Return data about a single movie by title to the user /movies/[Title] GET :title None A JSON object holding data about a single movie, structured like: 
						
{
	title: 'Soul',
	director: {
		name: 'Pete Docter',
		description: 'Peter Hans Docter is...',
		birthDate: 'October 9, 1968'
	},
	releaseDate: 'October 11, 2020',
	genre: {
		name: 'Animation',
		description: 'Animation is...'
	}
}
Return data about a genre by title /movies/genre/[Genre] GET :genreName None A JSON object holding data about the selected genre, structured like: 
						
{
	name: 'Animation',
	description: 'Animation is...'
}
Return data about a director by name /movies/directors/[Director] GET :directorName None A JSON object holding data about the selected director, structured like: 
						
{
	name: 'Pete Docter',
	description: 'Peter Hans Docter is...',
	birthDate: 'October 9, 1968'
}
Allow new users to register /users POST None A JSON object holding data just created by user, structured like: 
						
{
	name: 'John Smith',
	usersMovies: ['High School Musical'],
}
A JSON object holding data just created by user with newly added ID, structured like: 
						
{
	name: 'John Smith',
	usersMovies: ['High School Musical'],
	id: 1
}
Allow users to update their user info (such as their name) /users/[ID] PUT :id A JSON object holding data with user info, structured like: 
						
{
	name: 'John Smith',
	usersMovies: ['High School Musical'],
	id: 1
}
A JSON object holding data with updated user info, structured like: 
						
{
	name: 'John Smith Updated',
	usersMovies: ['High School Musical'],
	id: 1
}
Allow users to add a movie to their list of favorites /users/[id]/[movieTitle] POST :id/:movieTitle A JSON object holding data about movie to add: 
						
{
	name: 'John Smith',
	usersMovies: ['High School Musical'],
	id: 1
}
A message indicating whether movie was successfully added to user's list.
Allow users to remove a movie from their list of favorites /users/[id]/[movieTitle] DELETE :id/:movieTitle None A message indicating whether movie was successfully removed from user's list.
Allow existing users to deregister /users/[id] DELETE :id None A message indicating whether user was successfully deleted.
